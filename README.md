# Polynom

Компилятор для предметно-ориентированного языка (DSL) — калькулятора полиномов.

## Описание

Проект реализует интерпретатор/компилятор для языка, позволяющего выполнять арифметические операции над полиномами. Использует классический подход построения компиляторов с применением лексического и синтаксического анализаторов.

## Технологии

- **C++** — основная логика и структуры данных
- **Lex (Flex)** — лексический анализатор
- **Yacc (Bison)** — синтаксический анализатор
- **Make** — система сборки

## Структура проекта

```
polynom/
├── app/         # Основное приложение (парсер, лексер)
├── eval/        # Модуль вычисления полиномов
├── Makefile     # Сборка проекта
└── README.md
```

## Сборка

```bash
make
```

## Использование

После сборки запустите исполняемый файл и вводите выражения с полиномами:

```
> 2x^2 + 3x + 1
> (x + 1) * (x - 1)
```

## Поддерживаемые операции

- Сложение и вычитание полиномов
- Умножение полиномов
- Вычисление значения полинома в точке
